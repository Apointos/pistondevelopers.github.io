About Piston

Piston is a user friendly open source game engine written in Rust, started by Sven Nilsen.

Rectangle Example

{% highlight Rust %} use graphics::*; use piston::{Game, Gl}; pub struct App { ... } // your application data impl Game for App { fn render(&self, c: &Context, gl: &mut Gl) { c.rect(0.0, 0.0, 0.5, 0.5).rgb(1.0, 0.0, 0.0).fill(gl); } } {% endhighlight %}
